all(cb, lst)

All expects a callback and any number of expressions.

All expects a callback and any number of expressions. The expressions are evaluated and if all of them are true then the callback is executed.

Example: FN.all(() => { // Do something important. }, 1 === 1, 5 === 5);

Parameters:
Name Type Description
cb function

The callback to execute in the form: () => {...}
lst boolean

The arguments to all. All takes a variable number of arguments and processes them all as if they were an array.
Source:
  • fn.js, line 58
Returns:

The result of the callback or undefined if even one expression evaluated to false or the user didn't pass in a callback.

rest(cb, lst)

Rest expects a callback and any number of values.

Rest expects a callback and any number of values. The values are treated as if they were a list and the callback is executed with the all but the first element from this list passed in as an array as its only argument.

Example: FN.rest((remainingElements) => { remainingElements.forEach((element) => { console.log(element); }); });

take(cb, n, lst)

Take expects a callback, an index and any number of values.

Take expects a callback, an index and any number of values. The values are treated as if they were a list and the callback is executed with as many elements as could be take from the rest params up to the value of n.

Example: FN.take((elements) => { elements.forEach((element) => { console.log(element); }); }, 2, "Lions", "Tigers", "Bears");
